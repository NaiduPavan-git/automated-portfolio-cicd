About Me

I am a student interested in web development, cloud computing, and DevOps. This portfolio demonstrates how a simple static website can be deployed using an automated CI/CD pipeline.

About This Project

This project is called Automated Portfolio Website Deployment using CI/CD Pipeline. The main idea is to deploy this static site automatically using GitHub Actions. Whenever I push changes to the main branch, GitHub Actions:

  • Checks out the repository
  • Prepares the site files as an artifact
  • Deploys the latest version to GitHub Pages

This removes manual deployment steps and ensures the live site always reflects the latest code.
